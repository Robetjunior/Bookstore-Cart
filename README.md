# bookstore-cart

## Tecnologias utilizadas:
```
- VUE

- Javascript ES6

- SCSS

- NPM

- Webpack
```
## Como inicializar o projeto:

Clone  o repositório: 
```
 git clone : https://github.com/Robetjunior/Bookstore-Cart.git
```
Instale os pacotes
```
npm install
```
